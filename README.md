# 5-Day AI Agents: Intensive Vibe Coding Course With Google

This repo contains the colab assignments of [5-Day AI Agents: Intensive Vibe Coding Course With Google](https://www.kaggle.com/competitions/5-day-ai-agents-intensive-vibecoding-course-with-google/overview).

## Assignments Overview

*   **[🎒 Day 1 Assignment: Atmospheric-Controls](https://www.kaggle.com/competitions/5-day-ai-agents-intensive-vibecoding-course-with-google/discussion/708280)**
    A foundational project exploring basic environment controls and web interfaces built with AI agents.

*   **[🎒 Day 2 Assignment: .agents (Skills)](https://www.kaggle.com/competitions/5-day-ai-agents-intensive-vibecoding-course-with-google/discussion/708469)**
    A custom workspace directory containing specialized AI agent skills, security hooks, and paved-road configurations.

*   **[🎒 Day 3 Assignment: Skill_Tests and customer-support-agent](https://www.kaggle.com/competitions/5-day-ai-agents-intensive-vibecoding-course-with-google/discussion/708744)**
    A robust customer support AI agent implementation alongside comprehensive testing suites to evaluate specific skill executions.

*   **[🎒 Day 4 Assignment: ambient-expense-agent and shopping-assistant](https://www.kaggle.com/competitions/5-day-ai-agents-intensive-vibecoding-course-with-google/discussion/709165)**
    Advanced ADK agent workflows demonstrating background task processing for expense tracking, and a secure multi-tool shopping assistant with threat modeling and security gating.
